# Nest Template

## Environment Variables
1. Clone the .env.template file and rename it to .env.
2. Change environment variables.

## Lift the database
```bash
$ docker-compose up --d
```

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn start:dev

# watch mode
$ yarn watch:dev

# production mode
$ yarn start:prod
```
